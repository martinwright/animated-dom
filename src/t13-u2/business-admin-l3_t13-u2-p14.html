5 Gantt charts

Another planning tool is a Gantt chart, which is a useful and flexible device for illustrating a projectâ€™s progress. It makes the timeline very clear, can be simple or complicated, and can be adapted to match the complexity of the project. A Gantt chart illustrates the breakdown structure of the project in terms of cascading horizontal bars that show, for example:

  • start and finish dates
  • various relationships between project activities
  • how tasks track and influence each other.

The Gantt chart was developed by Henry Gantt, who was an American engineer and management consultant in the 1910s, and can be set up using specialist software or in Excel. As with PERT diagrams, the Gantt chart gives an overall view of the project with very clear references to timelines and how elements of the project impact each other.
