Delivering projects

6 Plan on a page

This is a simple tool that condenses the whole project down to one page of information. It is particularly useful to, for example:

  • help the project manager focus on the main points of a project – eg when making presentations to people outside the project
  • show stakeholders a very quick overview of the entire project – eg when applying for extra funding or resources from new sources
  • show the media the main points of the project – eg when launching a new product
  • use as a leaflet – eg when distributing information to the local community about a project that will affect or involve them.

Long and detailed reports and explanations are not always required, so a one-page version of the project can be an accessible and effective device for sharing the main points.
