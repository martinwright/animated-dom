Decision-making techniques

6 Gantt charts

Gantt charts are often used in project management to show activities against time. They are extremely useful when the problems are complex and interconnected, with various deadlines and targets that have an effect on each other, and are frequently used as planning tools.

You need to start by breaking down the overall problem into smaller tasks or activities. Each activity is represented by a bar, and its position and length illustrates the start and end dates.

Gantt charts enable you to see:

  • what the activities are
  • when activities and whole projects begin and end
  • how long each activity is scheduled to last
  • where they overlap, and by how much.

The charts can be larger and more complicated to show detail about a project. It is worth looking online to find examples of Gantt charts to see how complex they can be – using colours and graphics to denote different activities, and people’s roles and tasks.

When making decisions, they are a flexible tool that can be used to show all of the variables and how they need to work together. In the example above, for instance, the manager can see how the design process has to be finished by the end of March, so that it does not delay the implementation start date at the beginning of April. They might use this information when deciding about which designers and contractors they are going to use for the project, making sure that they can start and finish within the allocated time.
