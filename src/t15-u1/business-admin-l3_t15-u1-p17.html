Producing business documents

7 Version control

Where documents undergo edits and revisions, it is sensible to use a version control system from the beginning. Documents should be given a name and number, eg Report V1.docx.

Revisions to the document should be saved with the number incremented, eg Report V2.docx.

The author of the document should ensure the date the document is created or revised, and the version number, is identified on the first page and, where possible, is incorporated into the header or footer of the document. Some organisations use a draft/final version numbering system, allocating eg version 0.1, incrementing to 0.2, 0.3, etc, for draft documents; final documents using version 1.0, incrementing to 1.1 for any subsequent changes.
