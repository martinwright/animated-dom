6 Matching the development level to the style

Hersey and Blanchard make the matching development levels and styles simple through the letters and numbers applied to each as shown earlier. Therefore matching the development levels to the delegating styles is as follows:

  • The enthusiastic beginner D1, matches with the directing style S1.
  • The disillusioned learner D2, matches with the coaching style S2.
  • The capable but cautious D3, matches with the supporting style S3.
  • The self-reliant achiever D4, matches with the delegating style S4.

Therefore, through discussion between the manager and team member, once appropriate goals and objectives for the delegated task have been agreed, the development level for each task is identified, allowing the manager to match the appropriate style to the development level.
